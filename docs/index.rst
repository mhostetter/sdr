sdr
===

The :obj:`sdr` library is a Python 3 package for software-defined radio (SDR) applications.


Features
--------

- Filters: Infinite impulse response filters (:obj:`sdr.IIR`)
- Arbitrary resamplers: Farrow resampler (:obj:`sdr.FarrowResampler`)
